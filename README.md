# basic-ai

 bu projede sensörden alınan veriler kısmında json dosyasında sırayla sensörlerin verileri verilmiştir bunlardan birkaçtane girilip daha sonra bunlar arasındaki ilişkiyi yapay sinir ağları öğrenip daha sonra girirlen veriler ile ne zaman bozulacağını hesaplayan basıt bir ai modelidir.

"In this project, sensor data is provided sequentially in a JSON file; a simple AI model learns the relationships between these data using a neural network and then calculates when it will break down based on the data entered."
